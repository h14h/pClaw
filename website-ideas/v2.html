Get going in seconds

One env var. One command.

No complex setup. No dependency hell. No GPU provisioning. Just set your Vultr API key and run. You'll have a personal AI coding agent in under a minute.

pClaw is a single Go binary — lightweight enough to run on the cheapest VPS you can find.

terminal
# One command. That's it.
$ export VULTR_API_KEY="your-token"
$ go run .
That's all you need. One env var, one command, and you have a personal AI coding agent that remembers you.

Born from research. Built with care. Open source forever.

pClaw started as an experiment: what happens when you stop treating AI agents as disposable tools and start treating them as companions that grow with you? The answer turned out to be something surprisingly personal — an agent that remembers your codebase, adapts to your style, and gets better the more you work together. We believe the future of AI assistance isn't bigger models or flashier features. It's agents that know you.
